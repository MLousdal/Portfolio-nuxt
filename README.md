# 🌐 webTemplateVue
## 👓 Overview

webTemplateVue serves as a spring board to use when diving in to a new Vue project 🏊
- Setup with parcel for compiling 


## ⚙️ Compiling

```bash
🚀 Initialize node 
$ npm i

👨‍💻 Development
$ npm run dev

📯 Distribution
$ npm run build 
```
